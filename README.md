# 7星彩智能预测系统 v5.1

## 📊 项目概述
这是一个完整的七星彩（7-Star Lottery）数据分析和智能预测系统，包含两个核心模块：
1. **数据抓取模块** (`scrape_7star.py`) - 自动爬取七星彩历史开奖数据
2. **智能预测模块** (`predic_complex.py`) - 集成13种算法的号码预测系统

## 🔢 七星彩规则说明
- **位置1-6**: 范围 0-9（单位数）
- **位置7**: 范围 0-14（可为两位数：10, 11, 12, 13, 14）
- **总组合数**: 10×10×10×10×10×10×15 = 15,000,000 种可能

## 🕸️ 数据抓取功能 (`scrape_7star.py`)
- 自动从中国福利彩票官网爬取全部历史开奖数据
- 智能解析多种数据格式（支持位置7的两位数）
- 支持多种API接口和分页策略确保数据完整性
- 输出格式：
  - `7星彩_历史开奖号码.csv` - 完整版数据（含期号、日期、各位置号码）
  - `7星彩_号码序列.csv` - 精简版数据（仅期号和号码）
  - `raw_api_response.json` - 原始API响应（用于调试）

## 🧠 智能预测系统 (`predic_complex.py`)
集成 **13种算法** 的混合预测系统：

### 📈 统计方法 (6种)
1. **频率分析** - 基于历史出现频率
2. **热号冷号** - 识别近期热门和冷门号码
3. **马尔可夫链** - 基于状态转移概率
4. **加权近期** - 近期数据权重更高
5. **差值趋势** - 分析号码变化趋势
6. **模式匹配** - 寻找相似历史模式

### 🤖 机器学习/深度学习方法 (7种)
7. **贝叶斯概率模型** - Dirichlet-Multinomial后验推断
8. **随机森林** - 基于统计特征的分类器
9. **LSTM神经网络** - 3模型集成的长短期记忆网络
10. **GRU神经网络** - 3模型集成的门控循环单元
11. **GAN生成对抗网络** - 条件生成对抗网络
12. **强化学习** - Policy Gradient策略梯度算法
13. **TensorFlow Multi-Head Attention** - 基于注意力机制的序列模型

## 📋 系统特性
- **数据验证**: 自动验证历史数据范围正确性
- **训练/验证/测试划分**: 支持70%/15%/15%的数据集划分
- **集成投票**: 多算法加权投票产生最终预测
- **置信度评估**: 提供预测结果的置信度评分
- **详细分析**: 包含历史数据统计、概率分布、算法一致性分析
- **完整日志**: 自动生成详细的预测日志文件
- **模型缓存**: 支持模型检查点保存和加载，避免重复训练
- **数据增强**: 实现有效的扰动数据增强技术

## 🚀 使用方法
1. **抓取历史数据**:
   ```bash
   python scrape_7star.py
   ```

2. **运行智能预测**:
   ```bash
   python predic_complex.py
   ```
   - 系统会自动加载 `7星彩_历史开奖号码.csv`
   - 如无CSV文件，使用内置示例数据
   - 交互式设置训练参数（轮数、早停等）

## 📁 输出文件
- `logs/` 目录 - 包含详细的训练日志和预测结果
- `comprehensive_prediction_*.log` - 完整预测日志
- `checkpoints/` 目录 - 模型检查点文件（用于缓存训练好的模型）
- 各算法的独立训练日志文件
